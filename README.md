# student-grade-management-system
JavaFX-based Student Grade Management System implementing OOP concepts such as inheritance, interfaces, and data management.
This project is a desktop application built باستخدام Java and JavaFX to manage students, courses, and grades through a graphical user interface.

## Features
- Add and manage students (ID, name, department)
- Add and manage courses (course ID, name, credit)
- Assign grades to students for specific courses
- Display student and course information
- Calculate average grades
- Input validation and error handling

## OOP Concepts Used
- Inheritance (Person → Student)
- Encapsulation (private fields with getters/setters)
- Interfaces (custom interface for calculations)
- Comparable interface (sorting students)

## Technologies
- Java
- JavaFX
- Object-Oriented Programming

## Data Handling
- File-based storage (TXT/CSV)
- Data is loaded at startup and saved after updates
